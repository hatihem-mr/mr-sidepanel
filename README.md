## Installation

1. Download the latest version from Chrome Web Store (or load the extension manually)
2. Click the extension icon in your browser toolbar
3. Open a Muck Rack conversation in Intercom to get started

## How to Use

### Search Tab
1. Enter search terms in the search bar
2. Select your search categories (People, Outlets, Articles, etc.)
3. Click "Search" to see results
4. Click result cards to flip for more details

### Results History
- View your previous searches
- Star searches to add to favorites
- Click any search to run it again

### AI Analysis
1. Open a conversation in Intercom
2. Go to the AI Analysis tab
3. The extension will summarize the conversation
4. Get suggested replies and troubleshooting steps

### Settings
- Configure your API key for AI features (optional)
- Adjust extension preferences

## Requirements

- Chrome browser
- Access to Muck Rack
- (Optional) OpenAI API key for AI features
